# Senku Protocol

## Overview
A Next.js 15 web application for scanning Solana smart contracts. Features an AI-powered analysis interface using Google's Generative AI.

## Project Architecture
- **Real-time Data**: Integrated `useCryptoData` hook for live BTC/ETH/SOL prices and Ethereum gas tracking.
- **UI System**: Upgraded to 'Interstellar Cyber-Art' theme with custom shaders, glass-morphism, and neon-glow components.
- **Animation Engine**: Optimized Framer Motion implementation with spring-physics and liquid transitions.

## Tech Stack
- **Framework**: Next.js 15.1.7 with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS 4
- **UI**: Framer Motion for animations, Lucide React for icons
- **Blockchain**: Solana Web3.js
- **AI**: Google Generative AI

## Development
- **Dev Server**: `npm run dev` (runs on port 5000)
- **Build**: `npm run build`
- **Production**: `npm run start`
